# socchina_-
《基于CH32V307的稳像云台控制系统》


本作品基于沁恒CH32V307核心板，设计了一款面向智能拍摄与辅助科技场景的多模态交互三自由度稳像云台系统。系统通过摄像头实时采集图像，采用轻量化卷积神经网络实现手势识别与人脸检测，触发三自由度云台控制。创新性地集成语音指令控制模块与低功耗蓝牙通信模块，构建手势+语音+视觉的多模态人机交互体系，并采用增量式PID算法实现三轴舵机的精准控制。硬件上集成高扭矩舵机、OLED状态反馈模块、语音模块、低功耗蓝牙通信模块及独立电源，依托CH32V307的外设资源，通过多组UART串口实现识别指令与舵机控制的高效交互。创新点在于多模态指令映射、三轴协同控制、语音-视觉跨模态控制技术及低功耗边缘计算架构，适用于需要非接触控制的智能拍摄场景，以及益于特殊人群的辅助科技应用，兼具高实用性与可扩展性。
